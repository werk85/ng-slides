Die 3 Standbeine von AngularJS

DOM-basiertes erweiterbares Templating

Erweiter HTML um neue Funktionen: 
      <ul>
        <li 
          ng-repeat="user in users" 
          ng-click="selectUser(user)">
          {{user.name}
        </li>
      </ul>
Eigene Tags: 
      <markdown>*Hello* **world**!</markdown>

Keine aufwendige Datenmodell-Synchronisation

Der $scope dient als Klebstoff zwischen der Ansicht und der Logik: 
      $scope.users = [
        {id: 1, name: 'Bob'},
        {id: 2, name: 'Jim'}
      ];

2-Way Databinding

Strukturierte und testbare Anwendungen dank Dependency Injection (DI) 

      angular.module('app', [])

      .controller('SettingsController', [
        '$scope', 'dialog',
        function($scope, dialog) {
          ...
        }
      ])

      .factory('dialog', [
        '$http',
        function($http) {
          ...
          return someDialog;
        }
      ])
